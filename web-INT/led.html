Domů LED

Ovládání integrované LED diody

Ovládání diody spočívá v zapínání / vypínání pinu 13

void setup() { //zde se zapisuje to co se stane po spuštění arduina
pinMode(13, OUTPUT); // nastavení pinu 13 do módu OUTUPUT
}

void loop() { // zde se zapisuje to co se bude opakovat po dokončení části po spuštění

digitalWrite(13, HIGH); //pin 13 zapisuje 1
delay(100); // počkáme 100ms
digitalWrite(13, LOW); //pin 13 zapisuje 0
}
